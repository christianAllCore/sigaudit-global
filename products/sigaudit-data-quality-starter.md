# SIGAudit Data Quality Starter

## Espanol

SIGAudit Data Quality Starter es una ficha publica para organizar revisiones iniciales de calidad de datos GIS con datos de ejemplo.

| Entregable | Uso |
|---|---|
| Plantilla Excel | Registrar capas, tablas, campos, conteos, nulos y hallazgos. |
| Demo Python | Revisar CSV de ejemplo para detectar nulos, duplicados y valores inesperados. |
| Checklist GIS | Verificar estructura, dominios, relaciones y tipos de datos. |
| Resumen ejecutivo | Presentar estado, riesgos y acciones recomendadas. |

Principios: usar datos sinteticos, documentar reglas, mantener trazabilidad y no publicar informacion privada.

## English

SIGAudit Data Quality Starter is a public brief for organizing initial GIS data quality reviews with sample data.

| Deliverable | Use |
|---|---|
| Excel template | Register layers, tables, fields, counts, nulls and findings. |
| Python demo | Check sample CSV files for nulls, duplicates and unexpected values. |
| GIS checklist | Verify structure, domains, relationships and data types. |
| Executive summary | Present status, risks and recommended actions. |

Principles: use synthetic data, document rules, keep traceability and do not publish private information.
