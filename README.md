# SIGAudit Global

Public portfolio lab for GIS data quality, utility data auditing, migration QA/QC, automation and safe technical documentation.

## Español

Este repositorio será usado como laboratorio público y seguro para proyectos de portafolio relacionados con:

- GIS y ArcGIS
- calidad de datos
- auditoría de migraciones
- automatización con Python y C#
- reportes técnicos
- validación de datos de utilities
- documentación técnica

No debe contener contraseñas, tokens, datos reales de clientes, enlaces privados, rutas internas, código de empleadores, información médica privada ni investigación patentable.

## English

This repository is a public and safe portfolio lab for projects related to:

- GIS and ArcGIS
- data quality
- migration QA/QC
- Python and C# automation
- technical reporting
- utility data validation
- technical documentation

It must not contain passwords, tokens, real client data, private links, internal paths, employer code, private medical information or patent-sensitive research.
